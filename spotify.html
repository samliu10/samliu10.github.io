Spotify Playlist Creator

Python

Date: April 2019


This application uses Spotipy, the Python API for Spotify, to cluster playlists with a k-means clustering algorithm. The playlists are organized by analyzing song characteristics such as tempo, style, key, volume, time signature, etc. The application creates playlists based on a given number of clusters and groups songs found in a playlist specified by the user.

The algorithm does k-means clustering on a given set of data. The user can specify a desired number of clusters k, and the algorithm starts with choosing k random centroids. Each data point is assigned to the nearest centroid to create clusters in what is known as partitioning. The centroids are then recalculated, and the previous steps are repeated until convergence, the point at which the centroids stop changing.

The visual representation of the process shows the data set being used, the number of clusters, the number of times centroids have been recalculated, and whether or not the algorithm has converged. There is also an option to see an overlay representing the range of data in a particular centroid. The second image to the left shows the initial state of the algorithm after choosing 7 clusters, and the third shows what those clusters look like when the algorithm has converged.
