Monopoly

Game & GUI | OCaml | Fall 2020

Everyone knows the game Monopoly — it's the classic board game in which players exchange money and claim properties in the fight to be the last man who’s not bankrupt. But unlike the one you probably played growing up, this is a Cornell-themed Monopoly game written using OCaml and functional programming.

Each property on the default board is a location at Cornell with a corresponding color type, price, and rent. In addition, this program isn’t limited to a predefined board. Users can design a board themselves and import it into the game in JSON format. Player actions are directed through the command line, so it is possible to play the entire game on the command line. But for a more interactive game, just open the GUI to use it as a real-life game board. On every turn, the command line prints out information about the current player's assets, their position on the board, and all options for moves they can make. Players can roll, buy and sell properties, buy buildings on properties, forfeit, and end their turn. Just like in real Monopoly, players receive $200 for passing "Go", and they also may land in jail.
